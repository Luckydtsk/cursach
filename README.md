# Система учета проектных работ студентов

Небольшое веб-приложение на Flask для учета учебных проектов: проекты, задачи, команды, сдача работ и отчеты.

## Что умеет

- Авторизация по ролям: студент, преподаватель, администратор
- Управление проектами и задачами
- Назначение студентов в команды и на задачи
- Загрузка файлов по проектам и задачам
- Комментарии преподавателей
- Отчеты и лента активности

## Стек

- Python 3.13+
- Flask
- SQLite
- Jinja2 + CSS

## Быстрый запуск

1. Установить зависимости:
   - через pip: `pip install -r requirements.txt`
   - или через uv: `uv sync`
2. Инициализировать базу данных:
   - `python scripts/init_db.py`
3. Запустить приложение:
   - `python app.py`
4. Открыть в браузере:
   - [http://localhost:5000](http://localhost:5000)

## Структура проекта

- `app.py` — точка входа
- `config.py` — конфигурация
- `database.py` — работа с SQLite
- `routes/` — маршруты и бизнес-логика
- `templates/` — HTML-шаблоны
- `static/` — стили
- `scripts/` — SQL-скрипты и инициализация БД

## Примечания

- База хранится в файле `database.db`.
- Загруженные файлы сохраняются в папке `uploads/`.
